ℹ️ Comment ça fonctionne
Aucune configuration — Supabase est intégré directement dans le code, disponible sur tous tes appareils sans rien faire.
Sauvegarde automatique — chaque modification est envoyée après 3 secondes d'inactivité.
Chargement au démarrage — les données sont récupérées depuis Supabase à chaque ouverture.
